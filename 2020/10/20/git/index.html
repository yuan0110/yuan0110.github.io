Git

Hexo使用时遇到

unable to access 'https://github.com/zbydaily/zbydaily.github.io.git/': The requested URL returned error: 403

原因是切换git的，但是windows默认保留的是上一个账号的username和email，因此Hexo deploy失败

解决：

  • 控制面板 – > 用户账号 –> window 凭据 删除之前的就好了,再次上传时就会提示需要输入当前仓库的账号和密码
