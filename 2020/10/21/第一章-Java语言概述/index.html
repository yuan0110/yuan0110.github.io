第一章 Java语言描述

1.1 Java语言的产生及其特点

  1. 关键字： 网络计算模式，跨平台，

  2. JVM：跨平台，执行class字节码文件，可移植性

  3. write once, run anywhere

  4. 特点：简单，面向对象，分布式，健壮，安全（沙箱），可移植，多线程，动态..

1.2 Java运行环境与发开发环境

  1. JRE 运行别人的程序

  2. JDK 开发Java程序

  3. 过程：编写源代码，编译成字节码文件（二进制），运行java文件

  4. 运行脚本javac HelloWorld.java java HelloWorld

  5. jar： jar.exe将一些文件压缩成JAR文件，发布应用程序，解释器执行java -jar HelloWorld

第四章 面向对象程序设计

OOP：Object Oriented Programming 面向对象编程

特点：

* 封装
* 继承：Java不支持多重继承，子类只能有一个父类
* 多态：方法重载

class

类声明的格式：

类名的要求：规范是首字母大写，采用驼峰式写法

类的修饰符有三个：final, abstract,public(final和abstract不能修饰同一个类)

extends: 继承；只能有一个父类

implements：实现；可以试下多个接口类

类体

  1. 成员变量的修饰符

    • 默认访问修饰符：成员变量可以被同一个包（package）中的任何类访问

    • public：项目中的任何地方

    • protected：同一个package中的继承类

    • private：同一个class

    • static：类变量或者静态变量，同一个class的所有具体实例共享同一个变量。、

    • final:Java中的常量必须是类的成员，和static一起使用是不冲突的

    • transient：临时的，在默认状态下，所有的变量都是对象永久状态下的一部分；transient则指示JVM该变量不属于永久状态，主要用于实现不同对象的串行化功能

    • volatile:易变的，声明一个多线程共享变量

  2. 成员变量使用 ：类名.变量名

  3. 成员方法说明

    方法修饰符

    1. default：同一个包中的任何类
    2. public：被其他类访问
    3. protected：继承类
    4. private：只能由说明该方法的类访问
    5. static：静态方法，又叫类方法；无static修饰的叫实例方法。类方法是该类的所有对象共享的方法
    6. abstract：无方法体
    7. final：不能由子类改变
    8. synchronized：同步；实现线程同步
    9. native：本地，与本地计算机有关

    方法名应该是有意义的动词或者动词短语

  4. 实例化对象

    创建一个对象包括对象的声明，为对象分配内存空间和初赋值这三个步骤

  5. 方法重载

    方法的名字相同，但是参数不能相同
