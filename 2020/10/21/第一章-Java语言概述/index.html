Java

第一章 Java语言描述

1.1 Java语言的产生及其特点

  1. 关键字： 网络计算模式，跨平台，

  2. JVM：跨平台，执行class字节码文件，可移植性

  3. write once, run anywhere

  4. 特点：简单，面向对象，分布式，健壮，安全（沙箱），可移植，多线程，动态..

1.2 Java运行环境与发开发环境

  1. JRE 运行别人的程序

  2. JDK 开发Java程序

  3. 过程：编写源代码，编译成字节码文件（二进制），运行java文件

  4. 运行脚本javac HelloWorld.java java HelloWorld

  5. jar： jar.exe将一些文件压缩成JAR文件，发布应用程序，解释器执行java -jar HelloWorld

1.3 Java程序举例

  1. 分类：Application Applet Servlet

  2. 输入

    • BufferedReader ir = new BufferedReader(new InputStreamReader(System.in));

    • int a = ir.readline();

    • or Scanner cin = new Scanner(System.in); int a = cin.nextInt();int b = cin.nextInt;
